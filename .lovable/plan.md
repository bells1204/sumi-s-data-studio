

# Sumi Debnath — Premium Portfolio Website

A futuristic, recruiter-ready personal portfolio with glassmorphism design, smooth animations, and dark/light mode.

---

## Design System
- **Color palette**: Deep navy/slate background, electric blue & purple accents, neon glow effects
- **Style**: Glassmorphism cards with backdrop blur, gradient borders, soft shadows
- **Typography**: Clean modern fonts with gradient text highlights
- **Dark/Light mode** toggle with smooth theme transitions
- **Animated gradient background** with subtle floating particles/orbs

---

## Sections

### 1. Sticky Navbar
- Logo/name on left, section links on right
- Active section highlighting on scroll
- Dark/light mode toggle button
- Hamburger menu on mobile
- Glass-blur background effect

### 3. About Section
- Clean bio text with gradient accent highlights
- Key focus areas displayed as glowing tags/badges

### 8. Contact Section
